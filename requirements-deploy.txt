# Mallavaram — requirements for one-shot deploy on a new computer
#
# After cloning the repo, from the repo root:
#   .\Mallavaram-Workflows.ps1 -PlaywrightExportThenPushToGitHub
# (optional -AutoStartDevServerIfLocalhost if using localhost) — runs npm ci, playwright install, deploy-from-browser + publish-site.
# Or use "Download for deploy" in Configure then:
#   .\Mallavaram-Workflows.ps1 -MergeDownloadedBundleAndPushToGitHub
#
# Exact npm versions are pinned in package-lock.json — always commit the lockfile and use npm ci.

--------------------------------------------------------------------------------
1) Install on the machine first (not installable via npm)
--------------------------------------------------------------------------------

  Node.js     LTS 20.x or newer (includes npm). Required for all scripts.
              https://nodejs.org/

  Git         Required for npm run publish-site (commit + push to origin).
              https://git-scm.com/

  Windows     PowerShell 5.1 or later (Mallavaram-Workflows.ps1 uses #Requires 5.1).
              Included with Windows 10/11.

  Optional    Git remote + credentials configured for your GitHub repo (HTTPS or SSH).

--------------------------------------------------------------------------------
2) Configuration (not in npm — create locally)
--------------------------------------------------------------------------------

  .env.local  Copy from .env.example in the repo root. Set at minimum:
              MALLAVARAM_ADMIN_USER
              MALLAVARAM_ADMIN_PASSWORD
              Never commit .env.local (it is listed in .gitignore).

--------------------------------------------------------------------------------
3) npm packages — dependencies (package.json "dependencies")
--------------------------------------------------------------------------------

  @noble/hashes       ^1.7.1
  next                ^16.2.0
  react               18.2.0
  react-dom           18.2.0
  react-icons         5.2.1

--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
5) Playwright browsers (downloaded by deploy script, not npm packages)
--------------------------------------------------------------------------------

  Chromium    Installed with: npx playwright install chromium
              Mallavaram-Workflows.ps1 -PlaywrightExportThenPushToGitHub runs this unless -SkipPlaywrightChromiumInstall (-SkipPlaywrightInstall).
              Large download; requires network access on first run.

--------------------------------------------------------------------------------
6) Deploy pipeline commands (what the root script wraps)
--------------------------------------------------------------------------------

  npm ci | npm install
  npx playwright install chromium
  npm run deploy-from-browser
    -> npm run export-site-bundle   (tsx + playwright + dotenv)
    -> npm run publish-site         (tsx; uses git)
